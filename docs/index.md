# MQTTnet.AspNetCore.Routing Guides

This directory documents the generic MQTT routing programming model. The library owns topic matching, binding, dispatch, results, filters, diagnostics, and test helpers. Consuming applications own business topic contracts, authorization models, audit rules, schema policy, and compatibility strategy.
